# E2E Testing Framework (Playwright + TypeScript)
This project is an end-to-end testing framework built with Playwright and TypeScript for UI and API testing.
It simulates real-world test automation practices including API layer validation, UI workflows, and hybrid test scenarios.

## 📌 Project Purpose
The main goal of this project is to practice and demonstrate:

- UI test automation using Playwright
- API testing using Playwright request context
- Building a scalable test automation framework
- Applying Page Object Model (POM)
- Working with reusable utilities and test data
- Integrating UI + API testing flows

## Application Under Test
The tests are executed against a demo application:

- Automation Exercise (RealWorld App)
- URL: https://automationexercise.com

## 🛠️ Tech Stack
- Playwright
- TypeScript
- Node.js
- dotenv (environment variables)
- ESLint / Prettier
